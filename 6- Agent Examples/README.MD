# Agent Examples

This directory contains examples of AI agent architectures and environment setup.

## Projects

| Project Name | Description | Link to GitHub Dir |
|--------------|-------------|-------------------|
| VercelSDK | Multi-tool calling Customer Support Agent example built with JavaScript/Node.js & Vercel AI SDK. Features web search, function calling, and Retrieval Augmented Generation (RAG) workflows for increased answer accuracy. | [VercelSDK/](VercelSDK/) |
| ExtendedGeminiChat | Extended Gemini chat interface with function calling capabilities. Uses Google's Gemini API with support for custom functions and real-time token consumption tracking. | [ExtendedGeminiChat/](ExtendedGeminiChat/) |
| SimpleGeminiChat | Classic interactive chat interface with Google's Gemini API that tracks token consumption in real-time. Serves as a starter building block for other Agent examples. | [SimpleGeminiChat/](SimpleGeminiChat/) |
| Environment and LiteLLM Setup | Configuration notebook for LiteLLM integration and agent tooling setup. Demonstrates environment configuration for multi-provider LLM access. | [Environment and LiteLLM Setup.ipynb](Environment%20and%20LiteLLM%20Setup.ipynb) |
| PythonFunctionGenerator | Interactive notebook that generates Python functions based on natural language descriptions using AI, demonstrating code generation capabilities. | [PythonFunctionGenerator.ipynb](PythonFunctionGenerator.ipynb) |
| StarterAgentNotebook | Foundational agent implementation and development patterns. Provides a starting point for building custom AI agents. | [StarterAgentNotebook.ipynb](StarterAgentNotebook.ipynb) |
